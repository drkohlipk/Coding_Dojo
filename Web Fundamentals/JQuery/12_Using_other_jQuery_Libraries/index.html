Welcome to my playground!

The ninja/cat pic to the right uses the twentytwenty JQuery plugin, use the slider to change pics! The parallax inverted scrolling is due to the jInvertScroll Jquery plugin. If you scroll up and down, you'll see what I mean! If you go all the way to the right, you'll see a search bar with an animated entrance, this is all thanks to anime.js. Lastly, I've used Twitter's typeahead.js for autocomplete in the search bar. Have fun!
